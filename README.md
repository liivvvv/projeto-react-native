# app - gerenciador de tarefas (react native)

app em react native para a disciplina de tecnologias móveis e híbridas.

### como rodar o app

- na pasta do projeto, rode `npm install`
- depois, rode `npx expo start`
- aperte 'w' no terminal para abrir a versão web no navegador

**lembre-se de deixar a api rodando para que tudo funcione.**
